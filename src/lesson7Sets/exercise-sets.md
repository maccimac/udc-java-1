# Exercise: Sets

## The following steps will show you how to create a unique set from a list of phone numbers.

- [ ] Create a new package under the `src` folder named `setExample`
- [ ] Create a new class `SetExercise` in the `setExample` package
- [ ] Create a main function: `public static void main(String[] args){}`
- [ ] Create a list of numbers:
```
List<String> numbers = new ArrayList<String>();
```
- [ ] Add numbers to the list. Be sure to add multiple duplicate numbers.
- [ ] Create a set and add the list to its constructor:
```
Set<String> uniqueNumbers = new HashSet<String>
(numbers);
```
- [ ] Write a `for-each` loop to display the items in the set
- [ ] Right click on the `SetExercise` and select run
